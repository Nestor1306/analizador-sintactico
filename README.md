# analizador-sintactico

Un analizador sintactico hecho en Clojure

## Integrantes

- Emiliano Garcia

## Como correrlo

Estando en la carpeta del proyecto, ejecuta:

```bash
lein run resources/sample/<nombre_del_texto.txt>
```

```bash
open resources/sample/<nombre_del_texto.html>
```
